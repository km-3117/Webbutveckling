Animationer och Transitions (CSS)

Enkla exempel: transition (hover) och animation (@keyframes)

Transition (hover)

Transition används för att göra en mjuk övergång mellan två tillstånd när en egenskap ändras (t.ex. vid hover).

Hovra över den blå kvadraten för att se färgövergången och storlek förändringen.

.box { transition: background-color 0.3s ease; }

Animation (@keyframes)

Animationer använder @keyframes för att definiera steg över tid. Exemplet flyttar en cirkel från x=0 till x=200px.

Cirkeln rör sig fram och tillbaka automatiskt.

@keyframes moveRight { from { transform: translateX(0); } to { transform: translateX(200px); } }
